**Flappy Bird  (Pygame Edition)**

A remake of the classic Flappy Bird built with Python and Pygame 
Simple and fun arcade style game.

**Features**
- Smooth bird movement and gravity physics  
- Infinite random pipes generation  
- Score system.  
- funny sound effects.
- buttons (restart,fullscreen,mute,exit)

---

** languages used**
- Python   
- Pygame
